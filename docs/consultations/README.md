# 外部AIとの設計相談の記録

機能を設計する前に、外部のAI(ChatGPT等)に設計上の選択肢と落とし穴を相談した記録です。
⑧査読班(マージ前の差分レビュー、記録は `tools/state/reviews/`)とは別の、一回限りの相談です。

## 読むときの注意

- **相談相手のAIは何も決めていません。** 提案の採否はすべて運営者の判断で、
  サイトの原則(点数化しない・AIに評価の結論を出させない等)に照らして選り分けています。
- 相談相手の回答には、**確認されていない文献名や記憶ベースの記述が含まれえます**。
  サイト本文の根拠には、出典のルールを満たしたものだけを使います。
- 記録は原則そのまま残します(依頼文・回答全文・その後の実測を含む)。
