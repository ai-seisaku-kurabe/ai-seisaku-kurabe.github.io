# ⑨文献調査班の生の記録

このフォルダは、サイトの設計判断のために行った文献調査の**生の記録**です。
公開ページ「[先行研究と、この設計の根拠](https://ai-seisaku-kurabe.github.io/research.html)」は、
この記録から**出典のルール(実際にたどり着けたURLを持つ文献だけを本文で使う)を満たしたものだけ**を
選んで書かれています。

## 読むときの注意

- **未確認の記述を含みます。** 各ファイルの「未確認」節にある文献・主張は、一次資料に
  当たれていないものです。サイト本文の根拠には使っていません。
- **後に誤りと判明した記述も、そのまま残っています。** 記録は原則さかのぼって書き換えず、
  誤りは後続のファイル(例: `F_followup.md`)や research.html 本文で訂正します。
  例: 2026年公選法改正の記述(D_legal.md)は、その後の検証で条文の中身が未確認に戻されました。
- **これは発見層の資料であり、サイトの主張ではありません。** 調査の取り決めは
  `tools/agents/SURVEY_CHARTER.md`(このフォルダの `SURVEY_CHARTER_draft.md` はその草稿)。

## なぜ公開するか

このサイトは「判断の根拠を検証可能にする」ことを方針にしています。
整えた結論(research.html)だけでなく、その元になった生の調査(誤りや未確認を含む)も
残すことで、どこまでが確認済みでどこからが未確認かを第三者が追えるようにします。
